Present Value Calculation

The calculation performed by this application is calculating present value.
The definition of present value from Investopedia is "Present value (PV) is the current value of a future sum of money or stream of cash flows given a specified rate of return."
By setting the FV to $1,000,000, n to the years between current age and millionaire age, and assuming a rate of return of 7%, which is the historical average for the S&P 500, we can calculate the PV.
PV is the present value you would need in the market to reach $1,000,000.
